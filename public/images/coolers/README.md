## Important Notes

- All images are referenced in `/src/constants.ts`
- Image paths: `/images/coolers/{filename}`
- Recommended size: 800x600px or similar aspect ratio
- Use JPG format for better compression
- Ensure images are optimized for web (< 200KB per image)

## How to Add Images

1. Download or prepare product images
2. Name them according to the table above
3. Save them in this folder (`public/images/coolers/`)
4. Images will automatically load in the application

## Image Requirements

- **Format**: JPG, PNG, or WebP
- **Size**: 800x600px minimum
- **Quality**: High-quality product photography
- **Background**: Preferably white or transparent
- **File size**: < 200KB for optimal performance
